# 04-homework-LastFM
 
This repository include the secibd part of Homework 04: More API work from the LEDE Program in data journalism at Columbia Journalism School. More info can be found at: https://ledeprogram.com/.

The file uses the free LastFM API available at https://www.last.fm/api/ to complete a series of tasks involving rappers that have "Lil" in their name.
